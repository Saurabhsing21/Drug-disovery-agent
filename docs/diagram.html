⚙️ Execution Mode Sequential

Each collector node runs one after the other. LangGraph accumulates state using the Annotated[list, add] reducer — evidence items from all sources merge into a shared state list.

🕸️ LangGraph State

Defined in agent/state.py. The CollectorState TypedDict holds query, evidence_items, source_status, errors, and final_result.

🔀 Merge + Normalize

agent/normalize.py deduplicates evidence records by a stable hash of (source + target_id + evidence_type + raw_value) and normalizes scores to [0, 1].

⚡ Execution Mode Parallel

All connectors run simultaneously via asyncio.gather(). This is significantly faster than the sequential CLI flow, as network I/O operations are non-blocking.

🔌 Five MCP Tools Exposed

collect_depmap_evidence · collect_pharos_evidence · collect_opentargets_evidence · collect_literature_evidence · collect_evidence_bundle

📡 Transport

Uses mcp.server.stdio (stdio transport). The server reads from stdin and writes to stdout, making it plug-and-play with any MCP-compatible LLM orchestrator.
